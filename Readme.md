# Setup:

Head to the **main/** directory

create a virtual environment:

`$ make setup`

activate the environment:

`$ source venv/bin/activate`


install requirements:

`$ make requirements`


add he current directory to your `PYTHONPATH`:

`$ export PYTHONPATH="$PWD"`
